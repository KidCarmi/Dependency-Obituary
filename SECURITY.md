# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Dependency Obituary, please report it responsibly.

**Do NOT open a public GitHub issue for security vulnerabilities.**

Email: **security@orelsec.com**

We will acknowledge receipt within 48 hours and provide a timeline for a fix.

## Scope

- The web application at dependency-obituary.orelsec.com
- The GitHub App (dependency-obituary-bot)
- The CLI tool (bin/check.js)
- The API endpoints (/api/*)

## Out of Scope

- Third-party services (GitHub, Upstash, Vercel)
- Self-hosted instances operated by others
- Social engineering attacks
